typedef boost::iostreams::stream<boost::iostreams::basic_null_device<char, boost::iostreams::output> > tide::NullOStream

A null implementation of the std::ostream interface.

This std::ostream implementation ignores every write, effectively discarding the data into the ether. Use it for the verbose output of a Tide object to disable verbose information output.
